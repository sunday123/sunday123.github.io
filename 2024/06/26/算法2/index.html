1 1 0 1 0
2 1 0 0
3 1 0
4 1

abcba

01234

2个问题：

1为什么不用左下角？左下角是i>j,不方便理解，相当于（j->i）

2.为什么i不从上到下？比如要判断abcba（i,j）是回文，bcb(i+1,j-1）必然是回文，而(i+1,j-1）在下方。

  • abcba
  • 如果字符相等，相邻1个元素以内，必然是回文
  • 如果字符相等，比如第一个a和最后一个a，就判断bcb是否为回文。
